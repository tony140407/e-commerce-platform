## 專案說明

---

- 主要使用 `Vue.js` 和 `Vite` 架設而成
- 引入 `Bootstrap 5` 樣式
- css 使用 `Bootstrap 5` 結合 `Sass`
- 使用 `Font Awesome` 圖示
- 有後台管理系統

## 使用技術

- Vue3
- Vite
- Vue Component
- Vue Router
- Vue Axios
- Vee Validation
- Swiper
- Bootstrap 5
- Font Awesome
- SweetAlert2
- AOS
